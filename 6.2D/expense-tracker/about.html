About This Task

Distinction Task 6.2 - Custom Web Application

Note: If you are not currently up to date, you should skip this task and return to it once you are up to date with the Pass Tasks. Do not allow Credit or Distinction Tasks to delay you in keeping up with the unit's Pass Tasks.

To pass this task, you must:

  • Submit your code to Workspace 2.1
  • Submit your answer sheet to the assignments page in Canvas under Pass Task 2.1

Overview

At this stage, you should have enough understanding of interface design and development to start thinking about creating your own custom web application.

Purpose

Demonstrate that you can design and implement your own web application using Bootstrap and VueJS frameworks.

Submission Details

You must submit the following files (periodically):

  • The HTML code for your web application (concatenate multiple files into a single file for submission).
  • The JavaScript code for your web application (concatenate multiple files into a single file for submission).
  • Three pictures of your wireframe design supporting mobile portrait, mobile landscape, and desktop with the row-column and context labeling for each wireframe (photo or scan, if not electronically created).
  • A short design/usage document outlining what your web application does and how it works.

Instructions

You have now completed tasks related to all of the unitâ€™s learning outcomes and can work toward demonstrating these in your own web application. Aim to create something of around the combined complexity of the calculator, units, and units web applications.

Specifically, your web application should:

  • Demonstrate the use of context view grouping (row-column grid system).
  • Demonstrate the use of modules (controllers and config, maybe even custom directives and filters).
  • Demonstrate the use of arrays.
  • Demonstrate the use of directives and filters.
  • Support mobile-first design concepts (portrait, landscape, and desktop).
  • Demonstrate accessibility support, especially on input forms and tables.
  • Use appropriate HTML5 coding conventions (case, indentation).

For High Distinction, review the High Distinction Project document to meet additional requirements.
